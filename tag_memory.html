HTML 标签记忆

HTML 常见标签及其作用

标签类型 标签 作用
文档结构标签 <html> HTML 文档的根标签
<head> 包含文档的元信息
<body> 包含网页要显示的所有内容
<title> 定义文档的标题，显示在浏览器的标题栏
文本内容标签 <h1> - <h6> 标题标签，从 <h1> 到 <h6> 表示重要性递减的标题
<p> 段落标签
<span> 行内元素，用于在不改变文本结构的情况下对部分文本进行样式设置或脚本操作
<strong> 表示强调的文本，通常以粗体显示
<em> 表示强调的文本，通常以斜体显示
列表标签 <ul> 无序列表标签，列表项以项目符号显示
<ol> 有序列表标签，列表项以数字或字母等顺序显示
<li> 列表项标签，用于 <ul> 和 <ol> 中
链接与图像标签 <a> 超链接标签，用于创建指向其他网页或资源的链接
<img> 图像标签，用于在网页中显示图片
表格标签 <table> 表格标签
<tr> 表格行标签
<td> 表格单元格标签
<th> 表格表头单元格标签，通常以粗体居中显示
表单标签 <form> 表单标签，用于创建用户输入的表单
<input> 输入框标签，根据 type 属性可以是文本框、密码框、单选框等
<textarea> 多行文本输入框标签
<select> 下拉列表标签
<button> 按钮标签
多媒体标签 <video> 视频标签，用于在网页中嵌入视频
<audio> 音频标签，用于在网页中嵌入音频
其他标签 <div> 块级元素，用于划分网页区域，常作为布局容器
<br> 换行标签，用于强制换行
<hr> 水平分割线标签，用于在页面中插入一条水平分割线

<a> 标签

属性 取值 说明
target _self 默认，在当前窗口打开链接
target _blank 在新窗口或新标签页打开链接
target _parent 在父框架中打开链接
target _top 在整个窗口中打开链接，取消所有框架

<img> 标签

属性 取值 说明
width 具体像素值，如 200 图像的宽度，单位为像素
width 百分比值，如 50% 相对于父元素的宽度比例
height 具体像素值，如 200 图像的高度，单位为像素
height 百分比值，如 50% 相对于父元素的高度比例
alt 任何描述性文本 用于在图像无法显示时替代显示

<input> 标签

属性 取值 说明
type text 文本输入框
type password 密码输入框
type radio 单选按钮
type checkbox 复选框
type submit 提交按钮
type reset 重置按钮
type file 文件选择框
checked 对于 radio 和 checkbox 类型 写了表示默认被选中，不写则表示未选中

<video> 、 <audio> 标签

属性 取值 说明
autoplay autoplay 表示自动播放
autoplay 不写 表示不自动播放
loop loop 表示循环播放
loop 不写 表示不循环播放

<table> 标签

属性 取值 说明
border 具体数字，如 1 表示边框的宽度，单位是像素

<div> 、 <span> 标签

属性 取值 说明
class 自定义类名，如 container 可由字母、数字、下划线等组成
id 唯一标识符，如 main-content 命名规则与 class 类似
