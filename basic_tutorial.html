HTML 基础教程

HTML 基本概念

定义：HTML 即超文本标记语言，是用来描述网页的语言。

作用：通过各种标签来构建网页的结构和内容，使浏览器能够正确地显示网页。

HTML 基本结构

  • 文档类型声明： <!DOCTYPE html> ，告诉浏览器当前文档是 HTML5 类型的文档。
  • 根标签： <html> 标签是 HTML 文档的根元素，所有其他标签都包含在这个标签内。
  • 头部区域： <head> 标签内包含了文档的元数据，如文档标题、字符编码等。其中， <title> 标签用于设置网页的标题。
  • 主体区域： <body> 标签包含了网页的实际内容，如文字、图片、链接等。

基本标签

  • 标题标签： <h1> - <h6> ，用于定义不同级别的标题， <h1> 级别最高，字体最大。
  • 段落标签： <p> ，用于定义段落，段落之间会有默认的间距。
  • 换行标签： <br> ，是单标签，用于在文本中强制换行。
  • 水平线标签： <hr> ，用于在页面中插入一条水平线。
  • 链接标签： <a> ，用于创建超链接，通过 href 属性指定链接的目标地址。
  • 图片标签： <img> ，用于在网页中插入图片，通过 src 属性指定图片的路径。

标签属性

概念：标签可以拥有属性，属性为标签提供了额外的信息或功能。

示例： <a href="https://example.com" target="_blank">点击访问</a> 中， href 和 target 是 <a> 标签的属性。

特殊字符

在 HTML 中，一些字符具有特殊的含义，如果要显示这些字符本身，需要使用特殊字符实体。比如，空格用 &nbsp; 表示，小于号用 &lt; 表示，大于号用 &gt; 表示。

注释

HTML 中的注释以 <!-- 开始，以 --> 结束，注释内容不会在浏览器中显示，用于对代码进行解释说明。例如： <!-- 这是一个注释 --> 。
